# Food EAT 🍔

A simple and responsive restaurant landing page built using HTML, CSS, and JavaScript.

## Features

* Responsive navigation menu with mobile sidebar
* Hero section with:

  * Book Table button
  * Food Menu button
  * Rotating plate animation
* Why Choose Us section featuring:

  * Fast Delivery
  * Fresh Food
  * 24/7 Support
  * Special Offers
* Footer with:

  * Quick Links
  * Contact Information
  * Newsletter Subscription Form

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## JavaScript Functionality

The JavaScript code handles:

* Opening and closing the mobile sidebar menu
* Closing the sidebar when clicking outside (overlay)
* Closing the sidebar using the Escape key
* Automatically resetting the menu state on larger screens
